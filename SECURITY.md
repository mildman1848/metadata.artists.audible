# Security Policy

Deutsch: [SECURITY.DE.md](SECURITY.DE.md)

## Supported Versions

Only the current release line receives updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0.x | :x:                |

## Reporting a Vulnerability

Please use GitHub Security Advisories for private disclosure:

- https://github.com/mildman1848/metadata.artists.audible/security/advisories/new

If that path is unavailable, contact the maintainers privately and do not open a public issue first.
