# metadata.artists.audible

Deutsch: [README.DE.md](README.DE.md)

Kodi metadata scraper for Audible author and narrator information.

## Repository Structure

- `metadata.artists.audible/`: installable Kodi addon
- `scripts/build_release.py`: builds release ZIP files
- `scripts/publish_to_repo.py`: publishes the addon to `mildman1848.github.io`

## Local Validation

```bash
npm run validate
```
