# metadata.artists.audible

English: [README.md](README.md)

Kodi-Metadaten-Scraper für Audible-Autorinnen, Autoren und Sprecherinnen bzw. Sprecher.

## Repository-Struktur

- `metadata.artists.audible/`: installierbares Kodi-Addon
- `scripts/build_release.py`: baut Release-ZIP-Dateien
- `scripts/publish_to_repo.py`: publiziert das Addon nach `mildman1848.github.io`

## Lokale Validierung

```bash
npm run validate
```
